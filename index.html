CARDSダッシュボード 使い方ガイド

Googleスプレッドシートをデータベースに、GASをAPIとして動作するタスク管理ダッシュボードです。

タスクタブ — 日々のタスク管理
追加右上「＋タスク追加」ボタン → フォームに入力して保存 編集タスクカードの鉛筆アイコン or ステータスバッジをクリック 絞込み担当者・カテゴリ・ステータスのドロップダウン、またはテキスト検索 KPI総タスク / 完了 / 進行中 / 未着手 / 期限超過を上部カードで確認
WBSタブ — スケジュール・ECイベント確認
ガントチャートタスクの期限日をカレンダー上のバーで表示 ECカレンダー楽天（赤）・Amazon（黄）のイベント期間をWBS上部に表示 表示調整左上の日付で開始日を変更 / スライダーで列幅を調整
業務報告タブ — Slack投稿
手順日付を選択 → 「サマリー生成」で当日のタスクを集計 → 「Slack投稿」 スレッド同日に業務報告済みの場合、自動でスレッド返信になる
オートメーションタブ — 定期タスクの自動生成
テンプレートタスク名・カテゴリ・繰り返し条件を登録して定期生成 ECイベント連動繰り返しに「ECイベントごと」を選択するとイベントに合わせて自動スケジュール 日程ルール記載日 = イベント開始の5営業日前 / 期限 = 2営業日前 ×5,0日マラソン / スーパーSALE期間中で日付が5か0で終わる最初の日
管理者タブ — 初期パスワード: admin
カテゴリタスクのカテゴリと色を追加・編集・削除 担当者メンバーを追加・削除（タスクの担当者選択肢に反映） ECイベント楽天・Amazonのイベント期間を管理（WBSと自動化スケジュールに反映） PW変更管理者パスワードを変更（ブラウザのlocalStorageに保存）
